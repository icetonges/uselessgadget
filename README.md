# The Study-Hour Ledger

A Next.js (React) port of a static "study hour ROI" calculator — an illustrative model of how one hour of focused study compounds into future free time.

## Stack

- Next.js 15 (App Router)
- React 19
- Plain CSS Modules (no UI framework) — visuals ported 1:1 from the original static HTML
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy

Push to `main` and import this repo in [Vercel](https://vercel.com/new) — it auto-detects Next.js, no config needed.
